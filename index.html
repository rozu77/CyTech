Introduction

Trends in Front End Development are always changing -- what worked yesterday may not work today. This can make being a Front End Developer pretty challenging!


But what’s most challenging about being a Front End Developer is also what’s most exciting – you’re always learning new technologies.

Front End

The front end, or client side, of the web app is everything you see and interact with in the browser as a user of the application.


A Front End Developer is a generic term for someone who builds the user-facing parts of websites and applications using HTML, CSS and JavaScript.

Back End

The back end, or server-side of the web application, where the application’s data is saved, updated, manipulated, and sent back to the client side.


Back End Developers use languages like Java, Python, Ruby and PHP to build the behind-the-scenes functionality of a website, such as storing and retrieving user data or securely accepting and storing credit card numbers.

Javascript

Javascript is the first programming language that you'll likely to learn as Front End Developer.It is use in the Back end, Front end, mobile and even desktop. Generally, front end developers write javascript to the browser using the DOM or Document Object Model. The DOM is like the representation of your webpage or application thata allows the javascript and the browser to communicate. Each HTML element on your webpages is represented by the DOM and you can use the DOM to manipulate those elements.
